
# Insight Summary Iteration V0.52a-2 (Label - left menu)

This is a code bundle for Insight Summary Iteration V0.52a-2 (Label - left menu). The original project is available at https://www.figma.com/design/MF48waGJq8joA3tiTz1B6n/Insight-Summary-Iteration-V0.52a-2--Label---left-menu-.

## 개발 환경 실행 (Development)

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버는 `http://localhost:3000`에서 실행됩니다.

## 배포 (Deployment)

웹에서 다른 사람들도 볼 수 있도록 배포하는 방법은 **[DEPLOYMENT.md](./DEPLOYMENT.md)** 파일을 참고하세요.

### 빠른 배포 (Vercel 추천)

1. [vercel.com](https://vercel.com) 접속 및 로그인
2. GitHub repository 연결 (또는 파일 직접 업로드)
3. 자동으로 빌드 설정 감지됨
4. "Deploy" 클릭 → 완료!

더 자세한 배포 가이드는 `DEPLOYMENT.md` 파일을 확인하세요.
